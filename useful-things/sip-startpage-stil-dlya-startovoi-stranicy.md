# Sip StartPage - стиль для стартовой страницы

Вкладка «Новое окно» в браузере — это обычно либо пустота, либо реклама, либо стандартный набор иконок от Яндекса или Google. Скучно. А если сделать так, чтобы при открытии новой вкладки ты сразу попадал в удобную, красивую и полезную среду?

Для этого существуют **стартовые страницы**. И одна из них — **Sip StartPage**.

#### Что это вообще такое?

Sip StartPage — это простое и стильное расширение для браузера, которое заменяет стандартную страницу новой вкладки на твою собственную. Автор проекта — **bgibson72**, код открыт, лежит на GitHub, так что с безопасностью там порядок (ну, насколько это вообще можно сказать про дополнения).

{% hint style="danger" %}
<mark style="color:$danger;">**Доступно для Firefox (включая Android), скоро будет для Chrome, для Safari — сборка вручную**</mark>
{% endhint %}

{% embed url="https://addons.mozilla.org/ru/firefox/addon/sip/" %}

{% hint style="info" %}
#### Чем она полезна?

Тут без лишних слов, Sip StartPage даёт тебе чистый лист, который ты можешь заполнить чем угодно. Чаще всего на таких страницах размещают:

* **Быстрый доступ к закладкам** - чтобы не лазить по меню.
* **Поиск** - сразу с выбором поисковика.
* **Погода, время, цитаты** - для настроения.
* **Красивый фон** - хоть абстракция, хоть своя картинка.
{% endhint %}

{% hint style="danger" %}
<mark style="color:$danger;">**Минусы:**</mark> <mark style="color:$danger;"></mark><mark style="color:$danger;">Как и у любого дополнения, которые лезет в новую вкладку — иногда тормозит запуск браузера (если страница тяжёлая). Но Sip вроде лёгкий.</mark>
{% endhint %}
