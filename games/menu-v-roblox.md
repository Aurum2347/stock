# Menu в Roblox

**Описание**

Данный UI-скрипт позволяет вывести кастомное меню в **Roblox**, добавляя различные визуальные эффекты. Он вызывается с помощью специальной команды и работает только в **своём собственном плейсе**, а не в Roblox Studio.

**Как запустить меню?**

1. **Зайти в свой Roblox-плейс** (не в Roblox Studio).
2.  Открыть чат и ввести команду:

    ```
    /console
    ```
3. Переключиться во вкладку **Server**.
4.  Вставить и выполнить следующий код:

    ```lua
    require(100263845596551)("Nikolos19", ColorSequence.new(Color3.fromRGB(71, 148, 253), Color3.fromRGB(71, 253, 160)), "Standard")
    ```

**Что можно менять в команде?**

Единственное, что **рекомендуется изменять** — это **никнейм пользователя** (`"Nikolos19"`). Остальные параметры лучше оставить без изменений, чтобы меню работало корректно.

**Дополнительные возможности**

В меню есть **NSWF контент**, и чтобы его получить, следуй шагам:

1. Перейти в **Requires**.
2. Ввести `"Neko"`.

Наибольшее количество биндов пока что, это у — "Maid+".

**Побочные эффекты**

* **Снижение FPS** – чем дольше меню активно, тем сильнее начинает **лагать ПК**. Возможно, это связано с утечкой ресурсов или чрезмерной нагрузкой на систему.
