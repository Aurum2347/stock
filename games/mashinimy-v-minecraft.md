# Машинимы в Minecraft

Золотая эра майнкрафт-видео, примерно 2017 год, подарила нам эпичные истории и кинематографические шедевры. Сегодня таких видео снимают меньше, но страсть к творчеству жива.

Главная проблема — **где взять толпу игроков для массовки?**

На помощь приходит **Motion Capture mod**, который спасает одиночных авторов. Мод доступен для версий Minecraft с **1.16.5 по 1.21.5** (самая новая на момент 29.04.2025). Для старых версий есть мод Blockbuster, но в этой статье разберем именно Motion Capture.

## Как пользоваться Motion Capture

1. **Установка и предупреждение**\
   После запуска Minecraft с установленным модом на новых версиях (примерно с 1.21.4) может появиться предупреждение о нестабильности. Игнорируем его и движемся дальше.
2. **Подготовка сцены**\
   Создайте локацию для съемки: постройте декорации, настройте освещение, расставьте блоки. Когда все готово, начинаем запись.
3.  **Запись действий**\
    Введите команду:

    ```
    /mocap recording start
    ```

    Мод начнет записывать все ваши действия: движения, постройку, разрушение блоков и т.д.

    **Важно**: После записи NPC могут проходить сквозь блоки или "гулять по воздуху", если вы изменили местность во время записи.
4.  **Остановка записи**\
    Чтобы закончить, введите:

    ```
    /mocap recording stop
    ```

    Вам предложат два варианта:

    *   **Сохранить запись**:

        ```
        /mocap recording save [имя_реплея]
        ```
    * **Удалить запись**: Просто повторно введите `/mocap recording stop`.
5.  **Воспроизведение записи**\
    Чтобы запустить запись, используйте команду:

    ```
    /mocap playing start [имя_реплея] [имя_игрока] [URL_скина]
    ```

    Поля `[имя_игрока]` и `[URL_скина]` необязательны — можно оставить пустыми.

***

## Создание и воспроизведение сцен

1.  **Создание сцены**\
    Для начала создайте новую сцену:

    ```
    /mocap scenes add "название_сцены"
    ```
2.  **Добавление записи в сцену**\
    Чтобы прикрепить запись к сцене, введите:

    ```
    /mocap scenes addTo [название_сцены] [название_записи] [время_обрезки_начала] [имя_игрока] [тип_скина] [ссылка_на_скин_или_ник]
    ```

    * `[время_обрезки_начала]` — сколько секунд с начала записи обрезать (например, чтобы убрать ненужное).
    * `[тип_скина]` — выберите, использовать ссылку на скин или никнейм.
    * Поля `[имя_игрока]`, `[тип_скина]`, `[ссылка_на_скин_или_ник]` необязательны.
3.  **Запуск сцены**\
    Чтобы воспроизвести сцену, введите:

    ```
    /mocap playing start .[название_сцены]
    ```

***

## Полезные советы и гайды

* **Изучите гайды**: Подробные инструкции доступны на английском. Рекомендую ознакомиться с ними для глубокого понимания мода.
* **Тестируйте сцены**: Перед финальной записью проверьте, как NPC ведут себя в сцене, чтобы избежать багов с прохождением сквозь блоки.
* **Экспериментируйте со скинами**: Используйте разные скины, чтобы оживить массовку и сделать видео ярче.
