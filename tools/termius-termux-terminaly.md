# Termius / Termux - Терминалы

## <img src="../.gitbook/assets/image (83).png" alt="" data-size="line"> Termius — SSH-клиент для всех платформ

Вам реально скучно, и вы вдруг захотели проверить, насколько вы «прошарены» в компьютерах? Отлично — качайте **Termius**!

**Termius — это SSH-клиент для всех платформ** — Windows, macOS, Linux, Android и iOS.\
С его помощью можно подключаться к терминалу удалённого сервера, VPS или даже к локальным устройствам вроде Raspberry Pi — всё через безопасный протокол SSH.

{% embed url="https://termius.com/download/windows" %}

<figure><img src="../.gitbook/assets/image (85).png" alt=""><figcaption></figcaption></figure>

Основные возможности:

* Сохранение хостов и группировка по проектам
* Поддержка ключей, паролей и двухфакторной аутентификации
* Настройка цветовых схем, шрифтов и горячих клавиш
* Синхронизация между устройствами (при наличии аккаунта)

Termius особенно удобен, когда нужно управлять сервером с телефона или быстро подключиться к ПК из другой комнаты — главное, чтобы оба устройства были в одной сети.

### <img src="../.gitbook/assets/image (84).png" alt="" data-size="line"> Termux — Linux-терминал на Android

Вы знали, что Android — это, по сути, обёртка вокруг Linux? (Ну, если вы вообще слышали про Linux 😏)

**Termux — это терминал Linux прямо в вашем Android-устройстве**, и **root не нужен**!\
Конечно, суперправ у вас не будет, но это не мешает запускать кучу всего: локальный сервер, бота для Telegram, скрипты для Discord, качалки, хакерские утилиты и даже Python-проекты.

Короче, **Termux — это Linux в кармане**, без лишних заморочек.

<summary>Fun факт - Android сделан на базе Linux</summary>

Android изначально построен на **ядре Linux** — именно поэтому на нём возможно запускать терминальные утилиты, работать с файловой системой и даже устанавливать Linux-совместимые пакеты. Как отмечают разработчики Google, «Android построен на открытом Linux Kernel», а начиная с Android 11 используется LTS-версия ядра (4.19 / 5.4). Более подробно об этом можно прочитать в статье на Хабре: [«Android-смартфоны работают на Linux»](https://habr.com/ru/companies/ruvds/articles/548946/).

Это объясняет, почему такие инструменты, как **Termux**, работают на Android без эмуляции: они используют реальное Linux-ядро, просто без стандартного пользовательского окружения (GNOME[^1], [KDE ](#user-content-fn-2)[^2]и т.п.).

</details>

Установка пакетов выполняется командой:

```bash
pkg install имя_пакета
```

Популярные примеры:

* `neofetch` — отображает информацию об устройстве и ASCII-логотип
* `cmatrix` — имитирует «падающий код» из фильма «Матрица»
* `openssh`, `git`, `python`, `nano`, `wget` — и сотни других инструментов

> ⚠️ Из-за отсутствия root многие системные действия (например, запуск служб на портах ниже 1024) недоступны. Но для локального использования этого часто достаточно.

### Подключение к Termux через Termius

Если телефон и компьютер находятся в одной Wi-Fi-сети, можно управлять Termux с ПК через Termius:

1.  В Termux установите SSH-сервер:

    ```bash
    pkg install openssh
    sshd
    ```

    Сервер запустится на **порту 8022**.
2.  Узнайте локальный IP телефона:

    ```bash
    ifconfig
    ```

    (Или посмотрите в настройках Wi-Fi.)
3. В Termius на ПК создайте новый хост:
   * **Host**: IP телефона (например, `192.168.1.35`)
   * **Port**: `8022`
   * **Username**: результат команды `whoami` в Termux
   * **Password**: задайте через `passwd` в Termux
4. Подключайтесь — вы получите терминал телефона с удобной клавиатурой и большим экраном.

> 🔒 Такое подключение работает **только в локальной сети** и не открывает доступ из интернета.

### Зачем это нужно?

* Управлять телефоном с ПК, если экран сломан
* Протестировать скрипты в Linux-среде без установки ОС
* Использовать Android как лёгкий сервер (например, запустить веб-сервер через Python)
* Изучать командную строку и основы Linux в кармане

[^1]: **GNOME** — современный, минималистичный, удобный для новичков.

[^2]: **KDE (Plasma)** — мощный, гибкий, очень настраиваемый, выглядит "как Windows", но лучше 😄
