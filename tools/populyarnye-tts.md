# Популярные TTS

**TTS** (Text-to-Speech) — технология преобразования текста в речь. Примеры: голос в Google Переводчике или озвучка видео ботами.&#x20;

***

## Lazypy.ro

{% embed url="https://lazypy.ro/tts" %}

Сервис с множеством голосов. Голос **Wiseguy** часто встречается в смешных роликах по FNAF (DSaF), например, для Уильяма Афтона. Есть голос **Максим** для русского текста.

{% hint style="success" %}
**Плюсы**:

* Большой выбор голосов.
* Бесплатно, без строгих ограничений.
{% endhint %}

{% hint style="danger" %}
**Минусы**:

* Ограничение по символам: 540–3000 (зависит от языка).
{% endhint %}

***

## ElevenLabs

{% embed url="https://elevenlabs.io/" %}

Популярный сервис, часто используется в TikTok и YouTube, например, для озвучки животных. Поддерживает настройку голоса.

{% hint style="success" %}
**Плюсы**:

* Текст до 5000 символов.
* Настройка интонации и тембра.
{% endhint %}

{% hint style="danger" %}
**Минусы**:

* Лимит около 10 000 токенов в день.
{% endhint %}

***

## Better SAM

{% embed url="https://imrane03.github.io/better-sam/" %}

Сервис с одним голосом, но с настройкой тона и скорости. Используется в видео по _Faith: The Unholy Trinity_, _Ultrakill_, _Mandela Catalogue_.

{% hint style="success" %}
**Плюсы**:

* Настройка голоса.
* Открытый исходный код.
{% endhint %}

{% hint style="danger" %}
**Минусы**:

* Только английский язык.
* Один голос.
{% endhint %}

***

## FastHub

{% embed url="https://fasthub.net/" %}

Известен голосом **Whisper**, похожим на Габриеля из _Mandela Catalogue_. Выбирается голос Whisper, вводится текст, запускается озвучка.

{% hint style="success" %}
**Плюсы**:

* Без ограничений по длине текста.
* Настройка голоса.
{% endhint %}

{% hint style="danger" %}
**Минусы**:

* Качество озвучки может быть неровным.
{% endhint %}

***

## Silero TTS (Тг бот)

**Ссылка**: @silero\_voice\_bot\
Telegram-бот с голосами, напоминающими персонажей из _S.T.A.L.K.E.R._ (Бандит) или _Portal_ (GLaDOS).

{% hint style="success" %}
**Плюсы**:

* Много голосов.
{% endhint %}

{% hint style="danger" %}
**Минусы**:

* Лимит 250 символов в день.
* Лучше работает с русским языком.
{% endhint %}

***

## ChatGPT

{% embed url="https://chatgpt.com/" %}

Сервис попал в список из-за бага: повторение символов создаёт необычную озвучку. Примеры можно найти в TikTok по запросу «ChatGPT озвучка». Про плюсы и минусы, тут думаю - лишнее.
