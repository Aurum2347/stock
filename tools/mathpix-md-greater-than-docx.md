# MathPix (MD -> Docx)

kdown — что это и где используется?

**Markdown** — это простой язык для оформления текста. Он понятный, лёгкий и используется для выделения заголовков, списков, ссылок и другого.

***

#### Примеры кода:

1. **Заголовки:**

```markdown
# Заголовок 1
## Заголовок 2
### Заголовок 3
```

2. **Выделение текста:**

```markdown
*Курсив*
**Жирный**
~~Зачёркнутый~~
```

3. **Списки:**

```markdown
- Элемент списка
- Другой элемент
  - Подпункт

1. Первый
2. Второй
```

***

### Где встречается Markdown?

Markdown используют:

* **ChatGPT**: для форматирования ответов.
* **Discord**: для выделения текста.
* **Telegram**: для стильных сообщений.
* **GitHub**: для README.
* **Obsidian**: для заметок.
* И т.д.

## Конвертация Markdown -> PDF/DOCX

Если вам нужно превратить текст в формате Markdown в документ PDF или DOCX, это можно легко сделать с помощью сервиса [Mathpix](https://accounts.mathpix.com/quickstart).

{% embed url="https://mathpix.com/" %}

Этот инструмент особенно полезен для **студентов** и **школьников**, которые готовят домашние задания через ChatGPT.&#x20;

Markdown невозможно на прямую вставить в Word или PDF. Mathpix помогает быстро решить эту проблему, конвертируя текст и формулы в удобный формат для отправки учителям.
