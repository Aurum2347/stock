# PowerToys - Утилиты для Windows

Microsoft PowerToys — это набор утилит, позволяющих опытным пользователям настраивать и оптимизировать работу Windows для повышения производительности. Пользователи могут переназначать различные сочетания клавиш, а также включать или отключать определенные функции по своему усмотрению.

{% embed url="https://learn.microsoft.com/en-us/windows/powertoys/" %}

{% hint style="info" %}
<mark style="color:blue;">Каждую из этих утилит можно включать или отключать в зависимости от потребностей пользователя, а также настраивать их поведение и сочетания клавиш для активации.</mark>
{% endhint %}
