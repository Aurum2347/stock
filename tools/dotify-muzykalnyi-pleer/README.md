# Dotify - Музыкальный плеер

## Dotify: Музыкальный плеер для меломанов

Dotify — это музыкальный плеер и экосистема, позиционирующая себя как решение для настоящих меломанов . Приложение стремится стать универсальным хабом для прослушивания музыки.

<figure><img src="../../.gitbook/assets/image (237).png" alt=""><figcaption></figcaption></figure>

### Возможности

Dotify предлагает функции, выходящие за рамки простого воспроизведения треков. Ключевой особенностью является возможность совместного прослушивания музыки с друзьями через создание виртуальных комнат.\
Пользователи могут импортировать плейлисты из различных сторонних сервисов. Плеер также позиционируется как платформа, объединяющая разные музыкальные сервисы, такие как SoundCloud, VK и Яндекс.Музыка, в одном интерфейсе.

Dotify представляет собой музыкальный плеер с акцентом на социальное взаимодействие и интеграцию с другими платформами. Основными функциями являются совместное прослушивание и импорт плейлистов.
