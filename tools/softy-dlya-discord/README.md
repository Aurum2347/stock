# Софты для Discord

## <img src="../../.gitbook/assets/image (143).png" alt="" data-size="line"> BetterDiscord

Если вы задумывались о расширении функционала Discord, то да, существуют сторонние программы, которые позволяют это сделать. Одной из таких является **BetterDiscord**.

### Что такое BetterDiscord?

**BetterDiscord** — это модификация для стандартного клиента Discord, предоставляющая возможности полной кастомизации. С её помощью можно устанавливать и создавать собственные плагины и темы для Discord, значительно расширяя функционал и изменяя внешний вид приложения.

### <mark style="color:green;">**Плюсы**</mark>

{% hint style="success" %}
**Открытый исходный код:** BetterDiscord является проектом с открытым исходным кодом, что позволяет сообществу вносить улучшения и обеспечивать прозрачность разработки.
{% endhint %}

{% hint style="success" %}
**Поддержка пользовательских тем:** Возможность изменять оформление приложения, устанавливая различные темы, что позволяет сделать интерфейс более привлекательным и персонализированным.
{% endhint %}

{% hint style="success" %}
**Поддержка плагинов:** BetterDiscord поддерживает плагины, работающие только на стороне клиента. Они добавляют функции, например встроенный переводчик или отключение игровой активности.
{% endhint %}

{% hint style="success" %}
**Плагин FavoriteMedia:** Предоставляет функции, отсутствующие в другом софте Discord.
{% endhint %}

### <mark style="color:red;">**Минусы**</mark>

{% hint style="danger" %}
**Нарушение условий использования Discord:** BetterDiscord не является официальной модификацией и может нарушать правила Discord, что [**может** ](#user-content-fn-1)[^1]привести к **блокировке аккаунта.**
{% endhint %}

{% hint style="danger" %}
**Проблемы с производительностью:** у некоторых BetterDiscord делает приложение медленным и тупым.
{% endhint %}

## Установка BetterDiscord

1. **Скачивание:**
   * Перейдите на [официальный сайт BetterDiscord](https://betterdiscord.app/) и скачайте установочный файл для вашей операционной системы.
2. **Запуск установщика:**
   * Запустите скачанный файл.
3. **Выбор опции установки:**
   *   Выберите опцию `Install BetterDiscord` и нажмите `Next`.<br>

       <figure><img src="../../.gitbook/assets/image (144).png" alt=""><figcaption></figcaption></figure>
4. **Выбор версии Discord:**
   *   Выберите версию Discord, на которую хотите установить BetterDiscord (например, [Discord PTB](#user-content-fn-2)[^2]), и нажмите `Install`.<br>

       <figure><img src="../../.gitbook/assets/image (145).png" alt=""><figcaption></figcaption></figure>
5. **Перезапуск Discord:**
   *   После завершения установки Discord автоматически перезапустится. Если этого не произошло, запустите его вручную.<br>

       <figure><img src="../../.gitbook/assets/image (146).png" alt=""><figcaption></figcaption></figure>
6. **Проверка установки:**
   *   Перейдите в настройки Discord и прокрутите вниз. Если установка прошла успешно, вы увидите новые разделы настроек, связанные с BetterDiscord.<br>

       <figure><img src="../../.gitbook/assets/image (147).png" alt=""><figcaption></figcaption></figure>

### Настройки BetterDiscord

После установки в настройках Discord появятся следующие разделы:

* **Settings (Настройки):**
  *   Содержит базовые параметры BetterDiscord.<br>

      <figure><img src="../../.gitbook/assets/image (148).png" alt=""><figcaption></figcaption></figure>
* **Updates (Обновления):**
  *   Позволяет проверять и устанавливать обновления для BetterDiscord и установленных плагинов.<br>

      <figure><img src="../../.gitbook/assets/image (149).png" alt=""><figcaption></figcaption></figure>
* **Custom CSS (Пользовательский CSS):**
  *   Позволяет настраивать внешний вид Discord с помощью пользовательских CSS-стилей.<br>

      <figure><img src="../../.gitbook/assets/image (150).png" alt=""><figcaption></figcaption></figure>
* **Plugins (Плагины):**
  *   Здесь вы можете управлять своими плагинами: открывать папку для их добавления, а также включать или отключать их по необходимости.<br>

      <figure><img src="../../.gitbook/assets/image (151).png" alt=""><figcaption></figcaption></figure>
* **Themes (Темы):**
  *   Аналогично разделу плагинов, но предназначен для управления темами оформления.<br>

      <figure><img src="../../.gitbook/assets/image (152).png" alt=""><figcaption></figcaption></figure>

## <img src="../../.gitbook/assets/image (177).png" alt="" data-size="line"> Vencord

Если вы искали более оптимизированную и простую модификацию для Discord, то **Vencord** — отличный выбор. В отличие от BetterDiscord, он обладает улучшенной производительностью и простотой использования, хотя и имеет свои ограничения.

### Что такое Vencord?

**Vencord** — модификация для Discord с уникальными плагинами, темами, удобным интерфейсом и хорошей оптимизацией в отличии от BetterDiscord.

### **Плюсы**

{% hint style="success" %}
**Оптимизация:** Vencord работает быстрее и стабильнее, чем BetterDiscord.
{% endhint %}

{% hint style="success" %}
**Удобство работы с плагинами:** Уже встроенный список плагинов позволяет включать и выключать их одним кликом.
{% endhint %}

{% hint style="success" %}
**Эксклюзивные плагины:** Некоторые плагины, такие как "Decord", доступны только для пользователей Vencord.
{% endhint %}

### **Минусы**

{% hint style="danger" %}
**Отсутствие полного исходного кода:** Код Vencord недоступен для проверки или доработки.
{% endhint %}

{% hint style="danger" %}
**Нельзя установить свои плагины:** Пользователи ограничены только встроенным списком плагинов.
{% endhint %}

{% hint style="danger" %}
**Риски нарушения правил Discord:** Использование Vencord может привести к блокировке аккаунта, если модераторы обнаружат модификацию.
{% endhint %}

## Установка Vencord

#### Шаги установки:

1.  Скачайте установочный файл **VencordInstaller.exe** с официального сайта: [vencord.dev](https://vencord.dev/download/)<br>

    <figure><img src="../../.gitbook/assets/image (178).png" alt=""><figcaption></figcaption></figure>
2.  Запустите установщик. В открывшемся окне выберите версию Discord, для которой вы хотите установить Vencord (например, **Discord PTB**)<br>

    <figure><img src="../../.gitbook/assets/image (179).png" alt=""><figcaption></figcaption></figure>
3.  Ниже выберите одну из следующих опций:

    * **Install** — для установки Vencord;
    * **Reinstall / Repair** — для переустановки или исправления проблем;
    * **Uninstall** — для удаления Vencord;
    * **Install OpenAsar** — дополнительная установка (необязательно).

    Для начала установки выберите **Install** и нажмите на кнопку.<br>

    <figure><img src="../../.gitbook/assets/image (180).png" alt=""><figcaption></figcaption></figure>
4.  После завершения установки вы получите уведомление об успешной инсталляции. Закройте установщик.<br>

    <figure><img src="../../.gitbook/assets/image (181).png" alt=""><figcaption></figcaption></figure>
5.  Перезапустите Discord. Если установка прошла успешно, в настройках появятся новые опции, связанные с Vencord.<br>

    <figure><img src="../../.gitbook/assets/image (182).png" alt=""><figcaption></figcaption></figure>

Основные настройки Vencord. Здесь ничего особенного, можно оставить параметры по умолчанию.<br>

<figure><img src="../../.gitbook/assets/image (183).png" alt=""><figcaption></figcaption></figure>

Раздел для управления встроенными плагинами. К сожалению, добавить свои плагины нельзя, но вы можете легко включать и выключать доступные.\

Раздел для работы с темами. Установка тем осуществляется вручную, их нужно скачивать и загружать в соответствующую папку.<br>

<figure><img src="../../.gitbook/assets/image (185).png" alt=""><figcaption></figcaption></figure>

На этой странице можно проверить наличие обновлений для Vencord и установить их при необходимости.<br>

Позволяет экспортировать настройки Vencord и сохранять их локально или импортировать их обратно.<br>
