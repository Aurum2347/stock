# LibreSprite - бесплатный Aseprite

В 2016 году разработчик популярного редактора для пиксель-арта **Aseprite** закрыл исходный код и сделал программу платной. Но сообщество вовремя сохранило последнюю открытую версию (ещё под лицензией GPLv2) и сказало: «А давайте-ка сделаем из неё вечно бесплатный проект». Так и родился **LibreSprite** — форк, который живёт своей жизнью и не просит денег.

{% embed url="https://libresprite.github.io/#!/downloads" %}

Проект полностью открыт, исходники лежат на **GitHub**. Скачиваний там — тысячи, народ пользуется активно, так что с безопасностью там порядок (но, как говорится, «на свой страх и риск» всё равно никто не отменял).

#### Касательно функционала

По факту LibreSprite делает почти всё то же самое, что и платный Aseprite, потому что вышел из одного кода. Некоторых узкоспециализированных плюшек вроде продвинутых палитр или последних экспериментальных фильтров - _**не будет**_. Но для 95% задач разницы не будет.

{% hint style="info" %}
#### 🖥️ Технические характеристики

* **Последняя версия на момент 22.02.2026:** **1.2** (пре-релиз)
* **Русский язык:** есть официально. В версии 1.2 добавили машинный перевод, в том числе на русский. Лежит в файлах `.json` - если что-то не нравится, можно поправить вручную.
* **Вес программы:** \~**28 МБ** в установленном виде
* **Кросс-платформенность:** **Windows, Linux, macOS**. Есть экспериментальные сборки под Android и даже версия, которая работает прямо в браузере (через Emscripten).
* **Разработчик:** не конкретный человек, а сообщество. Код открыт, любой может глянуть, что там внутри, или предложить правку.
{% endhint %}

{% hint style="warning" %}
#### ⚠️ Из минусов

LibreSprite развивается медленнее, чем платный оригинал. Новые фишки Aseprite сюда не залетают, потому что кодовая база разошлась. Но баги правят, стабильность держат. Это такая консервативная, но рабочая лошадка.
{% endhint %}
